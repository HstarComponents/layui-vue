# lv-ui
将Layui制作为Vue2组件。（UI一直是个痛，那就先基于现有的UI，先封装几套玩玩。）

对应的LayUI版本为：`1.0.9_rls`，[LayUI 源码仓库](https://github.com/sentsin/layui)

示例对应的Vue版本为：`2.2.4`，[Vue 源码仓库](https://github.com/vuejs/vue)

# How to Run?

```bash
# Clone the project
git clone 

# Install dependencies
npm i 

# Run the examples app
npm run dev
```
